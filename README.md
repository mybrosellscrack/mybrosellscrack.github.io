# Духовный дневник

Веб-приложение для записи и анализа духовных переживаний, грехов и жизненных ситуаций.

## Конфиденциальность

**Главный принцип:** все данные хранятся только на устройстве пользователя.

- Данные сохраняются в localStorage браузера
- Никакие данные не передаются на сервер
- При очистке кэша браузера данные могут быть потеряны
- Исходный код открыт — вы можете проверить его на GitHub

### Рекомендации по безопасности

1. **Регулярно делайте бэкапы** — экспортируйте данные в CSV-файл
2. **Подключите Яндекс Диск** — для автоматических бэкапов при каждом изменении
3. **Не используйте режим инкогнито** — в нём данные не сохраняются

## Возможности

- **Новая запись** — добавление греха или жизненной ситуации
- **Архив** — просмотр всех записей с возможностью редактирования и удаления
- **Подготовка к исповеди** — группировка записей по неделям
- **Ситуации** — отдельный раздел для анализа жизненных событий
- **Статистика слабостей** — отслеживание повторяющихся грехов
- **Резервное копирование** — экспорт/импорт данных в формате CSV
- **Автобэкап на Яндекс Диск** — автоматическое сохранение копии в вашем облаке

## Как это работает

### Типы записей

1. **Грех** — для подготовки к исповеди. Попадает в раздел "Подготовка к исповеди" и "Статистика слабостей"
2. **Ситуация** — для анализа жизненных событий. Не участвует в подготовке к исповеди

### Поля записи

- **Ситуация** (обязательно) — описание события
- **Контекст** (необязательно) — обстоятельства
- **Последствие** (обязательно) — к чему привело
- **Инсайт** (обязательно) — урок или вывод

### Резервное копирование

1. **Ручной экспорт** — кнопка "Экспорт CSV" сохраняет файл на устройство
2. **Автоматический бэкап** — подключите свой Яндекс Диск, и копия будет сохраняться при каждом изменении

## Техническая информация

### Файлы

- `index.html` — основная структура
- `styles.css` — стили
- `app.js` — логика приложения
- `auth.html` — страница авторизации Яндекс Диска

### Запуск локально

```bash
python -m http.server 8000
```

Откройте `http://localhost:8000` в браузере.
