# Духовный дневник

Статичный сайт с блогом и тестом «Индекс душевного здоровья». Работает полностью на стороне клиента — без сервера, базы данных и регистрации.

## Состав

- **`index.html`** — главная страница с переключением между блогом и тестом
- **`blog/*.html`** — страницы статей (открываются по ссылкам из блога)
- **`styles.css`** — глобальные стили (оформление, навигация, индекс)
- **`blog.css`** — стили карточек блога и страниц статей
- **`app.js`** — логика навигации, рендеринг блога и полный тест

## Индекс душевного здоровья

Триединая шкала (Чувства / Воля / Желания) — 30 вопросов, по 10 на каждый блок.

- **0–7** Зелёный — гармония
- **8–15** Жёлтый — умеренные трудности
- **16–23** Оранжевый — выраженные проблемы
- **24–30** Красный — тяжёлое состояние

Данные теста сохраняются только в `localStorage` браузера. На сервер ничего не передаётся. При очистке кэша результаты будут потеряны. История замеров — до 60 записей.

## Технологии

- HTML5 + CSS3 (Custom Properties, Flexbox, Grid, backdrop-filter)
- Vanilla JavaScript (ES6+, localStorage, делегирование событий)
- Адаптивная вёрстка (mobile-first, @media)

## Разработка

Сайт полностью статичный. Для локального просмотра достаточно открыть `index.html` в браузере или раздать через любой HTTP-сервер:

```bash
npx serve .
# или
python -m http.server 8000
```
