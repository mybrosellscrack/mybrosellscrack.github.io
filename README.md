# Temporary Web Version (iPhone-friendly)

This folder contains a standalone HTML/CSS/JS web version of the app.
It stores data locally in the browser using `localStorage`.

Run locally:
- Start a simple local server in this folder, then open the site on iPhone.
- The iPhone must be on the same network as the server.

Example:
- `python -m http.server 8000`
- Open `http://<your-pc-ip>:8000` in Safari.

Files:
- `index.html`
- `styles.css`
- `app.js`
